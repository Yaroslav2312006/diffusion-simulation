# Моделирование диффузии частиц
Программный продукт для моделирования броуновского движения методом случайных блужданий. Реализован в рамках учебной ознакомительной практики (вариант А-09).
## Возможности
- Моделирование движения N частиц в двумерном пространстве.
- Визуализация траекторий, гистограммы распределения, графика MSD.
- Сравнение с теоретическим законом диффузии.
- Экспорт результатов в CSV.
## Установка и запуск
### Локальный запуск
1. Клонируйте репозиторий
2. Установите зависимости
3. Запустите моделирование с параметрами по умолчанию (1000 частиц, 100 шагов, шаг 1.0)
4. Чтобы изменить параметры, используйте аргументы командной строки

git clone https://github.com/Yaroslav2312006/diffusion-simulation
